# Nützliches

determinister endlicher Automat (DEA) ist ein 5-Tupel

M = (Q,∑,δ,q0,F)


```
Q  = endliche Menge an Zuständen
∑  = endliches Alphabet z.B {0,1}, {0,1,...,9}
δ  = Übergangsfunktion Q x ∑ --> Q
q0 ε Q = Startzustand
F  = Teilmenge von Q, Menge der akzeptierenden Zustände
```
